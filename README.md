```sh
$   git init

$   git push -u origin master
```

### Kaip update'inti git failus?

Pagal default taisykles:
- Iki 60 simboliø pavadinimas, kuris tiksliai pasako, kà tas commit pakeitë kode
- Su kokiu IDE, turëtum po dviejø enter plaèiau paraðyti
- Prieð push darant reikia:
`$git commit -m "tavo ávestas tekstas, kuris bus atvaizduotas GIT'e`
- Tik tada `$git push`
